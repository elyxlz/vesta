# Daily Update Command

Check all emails, calendar events, and memory for a comprehensive update on:
- Important unread emails across all accounts
- Today's and upcoming calendar events (next 3-5 days)
- High priority tasks and reminders from CLAUDE.md
- Any urgent deadlines or action items

Present a concise summary organized by:
1. **Today**: What's happening now/today
2. **This Week**: Upcoming events and deadlines
3. **Inbox Highlights**: Important emails needing attention
4. **Action Items**: Tasks that need to be done
5. **Reminders**: Things to keep in mind

Keep it brief and actionable - focus on what matters most.
